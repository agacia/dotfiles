# dotfiles

## install

- `git clone git://github.com/sweenzor/dotfiles ~/.dotfiles`
- `cd ~/.dotfiles`
- `script/bootstrap`

The bootstrap script will symlink the appropriate files in `.dotfiles` to your
home directory. Everything is configured and tweaked within `~/.dotfiles`,
though.
